## Traceroute (Go)
* Linux & Windows(Exec `tracert`)
* IPv4/IPv6 OK

## Usage
```
go build -o <Executable File> main.go

## Example (Windows)
go build -o trace.exe main.go

## Example (Linux)
go build -o trace main.go
```
